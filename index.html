What is Css?

  1. CSS stands for Cascanding Style Sheets.

  2. CSS describes how HTML elements are to be displayed on screen, paper, or in other media.

  3. Css is not neccesary CSS saves a lot of work.It can cantrol the layout of multiple web pages all at once

  4. External stylesheets are stored in CSS files.

CSS can be added to HTML Documents in 3 ways:

  • inline - by using the style attibute inside HTML elements1
  • internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS file.
  • The most common way to add CSS, is to keep the styles in external CSS file. However, in this tutorial we will use inline and internal and internal style, because this is easier to demonstrate. and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that CSS class applies a style to multiple element ID. on the other hand, applies a style to one unique element, ID is also special in that you can use a special URL to link directly to an element and it's used by JavaScript.

In CSS selectors are used to target a special element or range of element on web page. Once an element has been targeted, a style or set of style can be applied to the element.

There is a wide range of selectors available. Two of the most commonly used are class and ID. Both are used to target elements to whitch a style should be applied.
